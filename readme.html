pattern 
=========
=> pattern Attribute is used to verify input value in a text.
=> Regular Expression is used to inputted value in pattern Attribute.
=> Regular Expression is built by Meta Characters and Quantifiers.
=========================================================
Meta Characters                                     Description
=================                                   ============
[A-Z]                  => Only A-Z char Allowed
[a-b]                  => Only a-z char Allowed
[0-9]                  => Only 0-9 char Allowed
[a-zA-Z] or [a-Z]      => Only a-z and A-Z char Allowed
[a,s,d,f,g,h,j,k]      => Only specific Char are Allowed
[^a,s,d,f,g]           => Excluding a,s,d,f,g remain all char Allowed
\d [0-9]                 
\w = [a-zA-Z0-9]       => Only a-z, A-Z and 0-9 are Allowed
[@#$&*]                => Specificied Special Symbols are Allowed
======================================================================
Quantifires
===========
{4}           => Exactly 4 char are Allowed
{4,10}        => min 4 and max 10

for..of and for..in 
==================
if we want to execute a group of statement for every elements 
present in given collection (string, array, objects) then we have to use
for..in or for..of loop
Syntax:
for (var variableName of collection)
{
    statement
    statement
    statement
}
Eg:
---
var name= "RAMA"
for (var data of name)
{
    alert(data)  //R A M A
}
for (var data of name)
{
    alert(data)  // 0 1 2 3
}
